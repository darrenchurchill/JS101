# JS101 Lesson 3
## Practice Problems: Easy 1
### Question 2

How can you determine whether a given string ends with an exclamation mark?

```js
let str1 = "Come over here!";  // true
let str2 = "What's up, Doc?";  // false
```

### Solution:
```js
str1.endsWith('!');  // true
str2.endsWith('!');  // false
```
